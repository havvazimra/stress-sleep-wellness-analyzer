# Stress & Sleep Wellness Analyzer 🌿

This project uses machine learning to predict **stress levels** and **sleep quality** using lifestyle and health data.

The system also provides **personalized wellness recommendations** based on predictions.

## Features

* Data cleaning and preprocessing
* Feature engineering
* Stress prediction using **MLPClassifier**
* Sleep quality prediction using **Random Forest**
* Lifestyle clustering using **K-Means**
* Feature importance analysis
* Interactive wellness interface using **ipywidgets**

## Machine Learning Models

* MLPClassifier (Stress prediction)
* RandomForestClassifier (Sleep prediction)
* KMeans (Lifestyle clustering)

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* ipywidgets
* Joblib

## Dataset Features

The dataset includes lifestyle and health factors such as:

* Age
* Gender
* Occupation
* Sleep Duration
* Physical Activity Level
* Daily Steps
* BMI Category
* Blood Pressure
* Sleep Disorder

## Project Goal

To explore how lifestyle factors influence **stress levels and sleep quality** and to demonstrate machine learning techniques for predictive wellness analysis.
